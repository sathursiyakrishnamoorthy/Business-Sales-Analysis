# Business-Sales-Analysis
A simple tool to calculate business margins and identify top-performing products.
# Product Profitability Analysis
This project demonstrates basic business analytics by calculating profit margins across different product lines.
##  Insights
- **High Margin Items:** Accessories like the 'Mouse' show the highest percentage margin.
- **Revenue Drivers:** High-ticket items like 'Laptops' drive the most total profit dollars.
